# NAVI-CORE AI

AI-powered pathfinding and navigation system using BFS, DFS, A* Search and Machine Learning.

## Features

- BFS Pathfinding
- DFS Pathfinding
- A* Search
- Maze Solver
- City Navigation
- Decision Tree Prediction
- Performance Comparison
- Interactive GUI

## Technologies

- Python
- Tkinter
- NetworkX
- Matplotlib
- Scikit-learn
